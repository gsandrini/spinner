# Spinner

A simple prize wheel that randomly selects a number.  
Tap **Go** to spin, and **Reset** to bring the wheel back to its starting position.

🌐 **[Live demo](https://gsandrini.github.io/spinner/)**

---

## Features

- Prize wheel with smooth spin animation
- Bilingual interface — switches between Italian and English
- Works offline — installable as a PWA
- Supports light and dark mode automatically
- No dependencies except Alpine.js

## Usage

### As a web app

Open the [live demo](https://gsandrini.github.io/spinner/) in any modern browser

### Install

The app is available as a **Progressive Web App (PWA)** and can be installed directly on your device

1. Open the [live demo](https://gsandrini.github.io/spinner/) in your browser
2. Look for the **install icon** in the address bar, or open the browser menu and select **"Install app"** / **"Add to Home Screen"**
3. Confirm the installation - the app will appear on your desktop or home screen and work just like a native app, even offline

## Tech stack

- [Alpine.js](https://alpinejs.dev) - Reactive UI (bundled locally)
- [Tailwind CSS](https://tailwindcss.com) - Styling (compiled locally)
